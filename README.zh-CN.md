## 📖 简介

**Three.js 技能** 是一组 AI 编码智能体技能，属于 [Full Stack Skills](https://github.com/partme-ai/full-stack-skills) 生态，由 [PartMe.AI](https://github.com/partme-ai) 维护。

本包包含 **18 个技能**，为构建 Three.js 3D 应用提供专业知识、工作流和参考文档。每个技能是一个独立的 `SKILL.md` 文件，AI 智能体按需加载。

## 📦 安装

```bash
npx skills add full-statck-skills/threejs-skills
```

或按需安装特定技能：

```bash
npx skills add full-statck-skills/threejs-skills --skill <skill-name>
```
